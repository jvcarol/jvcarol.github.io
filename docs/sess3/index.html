Bibliotecas Utilizadas

Bibliotecas Úteis:#

Neste tópico falaremos sobre as algumas bibliotecas úteis importadas no projeto e para que são ou podem ser utilizadas:

  • Reactstrap: fornece vários componentes baseados em bootstrap, que permitem trabalhar com responsividade de forma simples. Seguem os componentes disponíveis: lista de componentes.

  • React-Paginate: fornece um componente de paginação personalizável e simples.

  • React-Google-Maps: fornece um conjunto de componentes que envolvem as instâncias subjacentes da Google Maps JavaScript API v3.

  • React-Chartjs-2: fornece um conjunto de componentes em react baseados na biblioteca Chart.js.

  • React-Ionicons: fornece um super pacote de ícones SVG em forma de componentes do React.

  • React-Moment: fornece componentes em react baseasos na biblioteca Moment.

  • React-Bootstrap-Sweetalert: fornece um componente de alerta personalizável em formato de modal.
